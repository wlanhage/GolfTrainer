# GolfTrainer Admin Web (Next.js)

Desktop-focused admin portal for GolfTrainer.

## Scope in this first iteration
- Login flow against existing backend auth API.
- Admin dashboard (users overview).
- Full course creation + hole management wireframe for desktop.
- Missions CRUD wireframe against backend mission-admin endpoints.
- Caddy overview.

## Commands
- `npm install`
- `npm run dev`
- `npm run typecheck`

## Env
- `NEXT_PUBLIC_API_BASE_URL` (default `http://localhost:3000/api/v1`)
